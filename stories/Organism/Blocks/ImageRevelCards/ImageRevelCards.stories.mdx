# Content Card Component

The Image Revel Card component defines a card of information data with some items - Title, Image, Paragraph, and CTA Link. We are using the grid system here <code>assets/scss/grid.scss</code>. Molecules are used for creating this card.

<Story name="Image Revel Cards">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return (
      <ImageRevelCards
        data={caption.takeactiondata}
        takeActionHeading={caption.takeActionHeading}
        {...args}
      ></ImageRevelCards>
    );
  }}
</Story>
