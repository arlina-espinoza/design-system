# Parallax Gallery component

## Overview

The Parallax Gallery Component is created for showing gallery, designation, and image in particular section. It is a multi-usage component that creates are usually teasing some kind of content.

## Default Representation & various possible states

<Story name="Parallax Gallery">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return <ParallaxGallery 
        titleText={caption.titleText}
        descText={caption.descText}
        buttonText={caption.buttonText}
        galleryimg1={caption.galleryimg1}
        galleryimg2={caption.galleryimg2}
        galleryimg3={caption.galleryimg3}
        galleryimg4={caption.galleryimg4}
        {...args}
        ></ParallaxGallery>;
  }}
</Story>
