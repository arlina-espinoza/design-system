# Footer List items component
Footer List items components are used to present the site-related information links on the footer section.

-Two colors variation are created  Black and white colors.

<Story name="Footer List" parameters={{ backgrounds: { default: 'gray'}}}>
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return <FooterLists  data={caption.authordata} headerText={caption.headerText} ariacontrol={caption.ariacontrol} arialabel={caption.arialabel} {...args} ></FooterLists>;
  }}
</Story>

### Usage

- Take HTML from the HTML tab in canvas
- <u>To use SCSS include the following files in your implementation</u>
- Include the common SCSS files like normalize, variables from <code>assets/scss</code>
- Component SCSS file: <code>footerlists.scss</code>
- <u>To use CSS include the following files in your implementation</u>
- The compiled CSS file from <code>assets/css/style.css</code>
