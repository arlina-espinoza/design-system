# SDG Component

The SDG Component is created for showing the title, image on hover. It is a multi-usage component that creates are usually teasing some kind of content.

## Default Representation & various possible states

<Story name="SDG">
  {(args, { globals: { locale } }) => {
    const caption = getCaptionForLocale(locale);
    return (
      <Sdg data={caption.data} imgsrc={sdglinkimage} title={caption.title}></Sdg>
    );
  }}
</Story>
